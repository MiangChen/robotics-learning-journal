# Product Overview

This is a **robotics learning journal** focused on **embodied intelligence** and **task planning** research.

## Purpose
- Personal learning documentation for multi-robot systems research
- Academic literature review and analysis
- Technical notes on VLA (Vision-Language-Action), task allocation, and swarm intelligence

## Core Topics
- **Multi-Robot Task Allocation (MRTA)**: Taxonomy, algorithms, and optimization
- **Task Planning**: Market-based, explicit optimization, and learning-driven approaches
- **VLA Models**: RT-1, RT-2, OpenVLA, and related architectures
- **Formal Languages**: PDDL, HTN, LTL, RDDL for task specification

## Target Audience
- Graduate-level researchers and above
- Readers with basic understanding of LLMs and robotics

## Content Style
- Academic rigor balanced with accessibility
- Uses illustrative examples (e.g., "Lost Island Treasure Hunt" scenario)
- Chinese language documentation with English technical terms
- Progressive explanation: intuition → formalization → code

## Repository Structure
- Main documentation in Markdown format
- LaTeX essays for formal publications
- Python utilities for document processing
- PDF research papers for reference
